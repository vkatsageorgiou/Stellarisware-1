The Stellaris® "qs-scope" Application

The oscilloscope application which is flashed onto the EK board when delivered can be found in the "qs-scope" directory of the DriverLib source tree for your USB-capable Stellaris evaluation kit. The following topics provide information relevant to the application.

  • Oscilloscope signal connections.
  • Navigating the menu system.
  • Menu options.
  • Setting triggers.
  • Saving files.
  • USB host and device operation.
