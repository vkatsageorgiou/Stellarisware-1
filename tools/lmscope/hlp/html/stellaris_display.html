Display Control Options

The following options are available in the "Display" group of controls accessed via the qs-scope menu:

Channel 2 Enables or disables capture of data for the second channel of the oscilloscope. Disabling the second channel allows the maximum 1MSample/second capture rate to apply to channel 1 and will improve the resolution of the waveform for timebases at or below 10µS/division.
Timebase Sets the oscilloscope timebase. Values offered range from 2µS/division to 50mS/division.
Ch1 Scale Sets the vertical scaling for the channel 1 waveform. Values offered range from 100mV/division to 10V/division.
Ch2 Scale Sets the vertical scaling for the channel 2 waveform. Values offered range from 100mV/division to 10V/division.
Ch1 Offset Sets the vertical offset for the channel 1 waveform. Move the waveform in 100mV increments using the left and right direction controls. Holding the control pressed will cause the offset to change quickly.
Ch2 Offset Sets the vertical offset for the channel 2 waveform. Move the waveform in 100mV increments using the left and right direction controls. Holding the control pressed will cause the offset to change quickly.
