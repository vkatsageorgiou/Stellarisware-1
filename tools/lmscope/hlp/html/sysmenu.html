System Menu Options

The LMScope application provides several options accessible on the system menu by clicking the icon in the top left of the main oscilloscope window. These options are:

Save Bitmap... If waveform data has been received from the connected Stellaris EK board, this option will save the last waveform as a Windows bitmap (BMP) image file.
Save Spreadsheet... If waveform data has been received from the connected Stellaris EK board, this option will save the last captured data set as a comma-separated-value (CSV) file suitable for importing into spreadsheets.
Display Options... This option shows a dialog box allowing the content of the waveform display area to be customized. Options are provided to turn graticule, trigger position, trigger level and ground markers on or off.
LMScope Help... Selecting this option will show the help file you are currently reading.

Note that additional help sepcific to each control on the LMScope dialog box can be obtained by pressing the "F1" key when each control has the focus. Alternatively, press the "?" button in the top right of the LMScope dialog box then click on any control to see the same help information.
About LMScope... This option shows a dialog box which contains the application name and version number.
