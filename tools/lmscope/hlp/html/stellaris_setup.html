Setup Options

The following options are available in the "Setup" group of controls accessed via the qs-scope menu:

Captions This option controls display of  voltage scale and timebase information. If "ON", the current timebase and scales for each channel are overlaid on the left side of the waveform display area. If "OFF" this information is not shown.
Voltages This option controls display of  the measured minimum, maximum and mean voltage for each enabled oscilloscope channel. If "ON", the information is displayed on the right side of the waveform display. If "OFF" this information is not shown.
Grid The options controls whether or not the graticule lines in the waveform display area are shown.
Ground This option controls whether or not a dotted line is drawn representing the ground level for each channel.
Trig Level This option controls whether or not a red horizontal line is drawn representing the current trigger level selection.
Trig Pos This option controls whether or not a red vertical line is drawn representing the current trigger position.
Clicks This option allows the keypress clicks to be enabled or disabled.
USB Mode Thiis option controls whether the EK board is currently acting as a USB host or device. By default, the application configures the board as a USB device capable of connecting to a host PC system. To use a USB flash stick to save files, the mode must be switched to USB host. For more information, please see the "USB Host and Device Operation" topic.
