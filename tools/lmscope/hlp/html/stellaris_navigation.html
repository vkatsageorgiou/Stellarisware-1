Navigating The Menu System

All qs-scope controls and settings are accessed using the multi-direction switch labelled "NAVIGATE" on the EK-LM3S3748  board.  This control offers up, down, left, right and select functions in a single unit.  Rocking the control in the desired direction sends ``up'', ``down'', ``left'' or ``right'' messages to the application and pressing on the center sends a ``select'' message.

Controls and settings are arranged into groups by function such as display settings, trigger settings, file operations and setup choices.  These groups are accessed by pressing ``select'' to display the main menu.  With the menu displayed, use ``up'' and ``down'' to select between the available groups of controls.  When the desired group is highlighted, press ``select'' once again to dismiss the menu.

Controls from the currently selected group are shown in the bottom portion of the application display.  Use ``up'' and ``down'' to cycle through the controls in the group and ``left'' and ``right'' to change the value of, or select the action associated with, the control which is currently displayed.

For example, to set the oscilloscope to trigger on a rising edge, the following actions would be performed:

  1. Press "select" to show the qs-scope main menu.
  2. Press "up" or "down" until  the "Trigger" option is selected.
  3. Press "select" to dismiss the menu display.
  4. Press "up" or "down" until the "Trigger" control is displayed.
  5. Press "left" or "right" until trigger type "Rising" is displayed.
Obviously, if the trigger control group was already selected, the first three steps of this example would not be necessary.
