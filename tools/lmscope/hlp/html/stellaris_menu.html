Menu Options

For background information on general navigation in the qs-scope application, please see "Navigating the menu system".

The qs-scope application menu contains 5 groups of controls. Clicking on the group name in the left column will provide more detailed information on each of the controls available.

Display Controls in this group relate to waveform display and include timebase, scale and position adjustment for each channel and also a single- or dual-channel operation switch.
Trigger Controls relating to triggering and capture control are to be found in this group. The trigger channel, type, level and position are set here along with capture mode - continuous or one-shot.
Setup This group contains controls allowing customization of the waveform display area and selection of the current USB operating mode.
File Saving of bitmap or CSV files to either an installed microSD card or USB flash stick is controlled from this group.
Help Controls here allow simple connection help to be displayed and also provide the ability to "find" each channel's waveform if it is outside the current display area.
