Luminary Micro® Oscilloscope

Build 4781

Welcome to the Luminary Micro® oscilloscope help file. Here you will find information on the oscilloscope application running on your Stellaris® evaluation kit ("qs-scope") and also the companion Windows® application which connects to it via USB ("LMScope").

The oscilloscope application illustrates use of the Stellaris analog-to-digital converter to capture samples from one or two input channels at up to 1MSample per second. Software running on the ARM® Cortex-M3(tm) CPU performs trigger detection, manages the menu-based user interface, and renders the captured waveforms on the EK color display. After connecting the evaluation kit to a Windows PC via USB, the companion LMScope application can be run to provide full remote control and waveform display.
